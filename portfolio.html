Websites I've Made

Bookflix

This was my first site built with a team. I was a front-end developer along with Omaur Bliss, while Betiel Habtemariam and Jennifer Nelson worked on back-end development. This site allows users to search a movie, which will return a description of said movie. The user can then select this movie, which will return books either about or related to said movie. The user can then store books they wish to save in their library. This site makes use of several APIs, local storage, and session storage. Link to site here, and links to Omaur's, Jennifer's, Betiel's, and my Github pages here.

Dear Diary

Dear Diary was my second site that I built with a team. I was a back-end developer on this app, along with Bao Xiong and Melissa Hoang. Melissa Hoang also worked on front-end with Betiel Habtemariam and Asli Farah. Its purpose was to allow users to record their thoughts and ideas from a day in a secure site that they can access from any device. It makes use of express-server, session-storage, sequelize, and many other technologies. Link to site here, and links to Bao's, Melissa's, Betiel's, Asli's, and my Github pages here.

Random Password Generator

This website is from an assignment where I was given an existing site and had to modify it to make it work. I used Javascript to successfully accomplish this task. It gave me a better understanding of Javascript as a whole, as I made use of several new concepts such as 'if' statements and pulling from arrays. With this new knowledge, I was able to make a functioning random password generator. Link to website here.

Coding Quiz

This website was built from scratch. It's a quiz about dog trivia, with a functioning timer and a responsive layout. The user can also submit their initials with their time at the end of the quiz, which will then be saved to local storage. This site also made use of dynamically generating new screens from arrays upon button press. Link to site here.
